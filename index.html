What is a sensory substitution device?

Sensory substitution is a technique whereby one sense (e.g. vision) is artificially re-routed through another (e.g. hearing). This is usually done when a sense, typically vision, is lost. A sensory substitution device (SSD) is a tool which does the re-routing.
